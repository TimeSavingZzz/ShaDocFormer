## 七、重要注意事项

### DocDeshadower 分辨率限制
DocDeshadower 的 Laplacian Pyramid + ReflectionPad 结构要求输入至少 **512×512**。384×384 会在深层 ReflectionPad 时报错。训练 DocDeshadower 时需设置 `--res 512`：

```bash
python train_compare_models.py --model docdeshadower --epochs 300 --res 512
```

其他模型均可使用 384×384。

### 关于预训练权重
**不需要下载预训练权重。** 文档阴影去除领域所有对比模型均从零在 RDD 上训练。使用 Pre-trained on ImageNet/Denoising 的权重反而会造成不公平对比，审稿人会质疑。

- DocDeshadower (511K, 512×512): ~0.18h/epoch
- BEDSRGenerator (41.8M): ~0.35h/epoch

## 八、Resume 训练

checkpoint 包含 optimizer state，支持 resume：

```python
# 在 train_compare_models.py 中加载 checkpoint:
ckpt = torch.load('experiment_results_rdd/full_300ep/textaware_epoch100.pth')
model.load_state_dict(ckpt['model'])
optimizer.load_state_dict(ckpt['optimizer'])
start_epoch = ckpt['epoch']  # 从第 101 轮继续
```

需在脚本中添加 `--resume` 参数支持。目前脚本从 epoch 1 开始训练，如需 resume 功能可以快速添加。
