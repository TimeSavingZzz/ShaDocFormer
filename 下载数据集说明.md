# RDD数据集下载说明

## 数据集介绍

RDD (Real Document Dataset) 来自 CVPR 2023 论文 *"Document Image Shadow Removal Guided by Color-Aware Background"*。

- 训练集: 4371对图像
- 测试集: 545对图像
- 总计: 4916对

## Google Drive 下载链接

🔗 **https://drive.google.com/drive/folders/1nS-p9qKCsFjOFyzq5q2m-Dq-PJVmLfNc**

## 下载方法

### 方法一: 浏览器手动下载（推荐）

1. 使用VPN/代理访问上述Google Drive链接
2. Google Drive文件夹结构:
   ```
   RDD/
   ├── train/
   │   ├── shadow/    ← 阴影图像 (4371张)
   │   └── back_gt/   ← 无阴影真值 (4371张)
   └── test/
       ├── shadow/    ← 阴影图像 (545张)
       └── back_gt/   ← 无阴影真值 (545张)
   ```
3. 右键点击每个子文件夹，选择"下载"（Google会自动打包为ZIP）
4. 下载后解压到对应目录

### 方法二: gdown命令行（需要稳定网络）

```bash
# 安装gdown
pip install gdown

# 设置代理（如果需要）
set HTTP_PROXY=http://你的代理地址:端口
set HTTPS_PROXY=http://你的代理地址:端口

# 下载整个RDD文件夹
gdown --folder "https://drive.google.com/drive/folders/1nS-p9qKCsFjOFyzq5q2m-Dq-PJVmLfNc" -O "./dataset/RDD/"
```

### 方法三: 运行下载脚本

```bash
python download_rdd.py
```

## 目录重组

下载完成后，需要将Google Drive的目录结构转换为ShaDocFormer期望的格式：

Google Drive结构 → ShaDocFormer结构:
- `train/shadow/` → `dataset/RDD/train/input/`
- `train/back_gt/` → `dataset/RDD/train/target/`
- `test/shadow/` → `dataset/RDD/test/input/`
- `test/back_gt/` → `dataset/RDD/test/target/`

或者直接运行以下Python脚本自动重组:

```python
import os
import shutil

base = "dataset/RDD"
mapping = {
    "train/shadow": "train/input",
    "train/back_gt": "train/target",
    "test/shadow": "test/input",
    "test/back_gt": "test/target",
}

for src, dst in mapping.items():
    src_path = os.path.join(base, src)
    dst_path = os.path.join(base, dst)
    if os.path.exists(src_path):
        os.makedirs(dst_path, exist_ok=True)
        for f in os.listdir(src_path):
            shutil.move(os.path.join(src_path, f), os.path.join(dst_path, f))
        print(f"Moved {src} -> {dst}")
```
