#### (6) 训练循环核心 (65-87行)
每一轮的训练步骤：
```python
for epoch in range(1, 301):
    model.train()
    for _, data in enumerate(tqdm(trainloader)):
        inp = data[0].contiguous()   # 原始带阴影图 (RGB)
        gray = data[1].contiguous()  # 灰度版阴影图 (单通道)
        tar = data[2]                # 无阴影真值图 (RGB)

        optimizer_b.zero_grad()
        res = model(gray, inp)       # 前向传播

        loss_psnr = criterion_psnr(res, tar)           # MSE损失
        loss_ssim = 1 - structural_similarity_index_measure(res, tar, data_range=1)  # SSIM损失

        train_loss = loss_psnr + 0.2 * loss_ssim        # 组合损失 ★

        accelerator.backward(train_loss)                # 反向传播
        optimizer_b.step()                               # 参数更新
    scheduler_b.step()                                   # 学习率衰减
```

**关键设计 —— 组合损失函数**:
```
Total Loss = MSE(res, tar) + 0.2 × (1 - SSIM(res, tar))
```
- MSE: 保证像素级重建精度
- 1-SSIM: 保证结构相似性（SSIM越大越好，所以1-SSIM越小越好）
- 权重0.2: SSIM损失的权重较低，以像素级重建为主

#### (7) 验证与模型保存 (89-131行)
```python
if epoch % opt.TRAINING.VAL_AFTER_EVERY == 0:  # 每1个epoch验证一次
    model.eval()
    # 计算 PSNR, SSIM, RMSE
    for _, data in enumerate(testloader):
        with torch.no_grad():
            res = model(gray, inp)
        res, tar = accelerator.gather((res, tar))
        psnr += peak_signal_noise_ratio(res, tar, data_range=1)
        ssim += structural_similarity_index_measure(res, tar, data_range=1)
        rmse += mean_squared_error(res*255, tar*255, squared=False)

```

**验证指标（均在1-1数据范围计算）**:
- **PSNR** (Peak Signal-to-Noise Ratio): 峰值信噪比，越高越好
- **SSIM** (Structural Similarity): 结构相似度，越接近1越好
- **RMSE** (Root Mean Square Error): 均方根误差，越低越好（在0-255范围计算）
- **模型选择策略**: 以验证集RMSE最低的epoch作为最佳模型保存

## 四、模型架构

### 4.1 主模型 — `models/model.py`

```python
class Model(nn.Module):
    def __init__(self):
        self.mask = ThresholdFormer(num_trans_blocks=3)   # 阴影检测模块
        self.refine = Refiner()                             # 图像修复模块

    def forward(self, bin_x, x):
        mask = self.mask(bin_x)           # Step 1: 检测阴影区域
        x_res = torch.cat((x, mask), dim=1)  # Step 2: 拼接原图+阴影掩膜
        res = self.refine(x_res)           # Step 3: 修复阴影区域
        return res
```

**数据流**:
```
灰度图(bin_x, 1通道) → ThresholdFormer → 阴影掩膜(2通道)
                                                    ↓
                                          拼接 [原图(3通道) + 掩膜(2通道)] = 5通道
                                                    ↓
                                           Refiner → 无阴影输出(3通道)
```

**两条输入路径**:
1. `bin_x` (灰度图): 仅用于阴影检测（ThresholdFormer），降低计算量
2. `x` (原图): 用于最终修复（Refiner），保留颜色信息

### 4.2 阴影检测模块 — `models/mask.py`

`ThresholdFormer` 是论文的核心创新模块，结合了**传统图像处理（Otsu阈值法）和深度学习（Transformer）**。

```python
class ThresholdFormer(nn.Module):
    def __init__(self, num_trans_blocks):
        # 特征提取头: Conv2d(1→4→16)
        model = [nn.Conv2d(1, 4, 3, padding=1),
                 nn.InstanceNorm2d(4),
                 nn.LeakyReLU(),
                 nn.Conv2d(4, 16, 3, padding=1),
                 nn.LeakyReLU()]

        # Transformer编码器: num_trans_blocks个TransformerBlock
        for _ in range(num_trans_blocks):
            model += [TransformerBlock(dim=16, num_heads=1, ...)]

        # 输出头: Conv2d(16→4→1)
        model += [nn.Conv2d(16, 4, 3, padding=1),
                  nn.LeakyReLU(),
                  nn.Conv2d(4, 1, 3, padding=1)]

        self.model = nn.Sequential(*model)

    def forward(self, bin_x):
        # 1. Otsu阈值法生成初始掩膜
        hist = torch.histc(bin_x.view(-1), bins=256, min=0, max=1)
        pmf = hist / hist.sum()                         # 概率质量函数
        cdf = torch.cumsum(pmf, dim=0)                  # 累积分布函数

        cumulative_means = torch.cumsum(pmf * torch.arange(0, 256).cuda(), dim=0)
        total_mean = cumulative_means[-1]
        inter_class_variances = (total_mean * cdf - cumulative_means) ** 2 / (cdf * (1.0 - cdf) + 1e-12)
        otsu_threshold = torch.argmax(inter_class_variances) / 255  # 最佳阈值

        otsu_mask = torch.ones_like(bin_x)
        otsu_mask[bin_x > otsu_threshold] = 0           # 亮区(阴影)设为0

        # 2. Transformer学习精细化掩膜
        trans_mask = torch.sigmoid(self.model(otsu_mask))

        # 3. 拼接两种掩膜输出
        res_mask = torch.cat((otsu_mask, trans_mask), dim=1)  # 2通道输出
        return res_mask
```

**ThresholdFormer的设计哲学**:
1. **Otsu阈值**: 传统方法，利用灰度直方图的两类方差最大化找到分割阈值。提供初始粗糙阴影区域估计。
2. **Transformer优化**: 3个TransformerBlock对Otsu掩膜进行精细化，学习Otsu无法捕捉的复杂阴影边界和渐变区域。
3. **双掩膜拼接**: 输出2通道掩膜（Otsu掩膜 + Transformer掩膜），让后续Refiner可以使用两种信息。

**TransformerBlock结构**:
```
输入 → LayerNorm → NextAttentionZ(行注意力+列注意力) → 残差连接
     → LayerNorm → FeedForward(双门控FFN) → 残差连接 → 输出
```

**NextAttentionZ**: 轴向多头自注意力机制
- Row Attention: 沿宽度方向做自注意力
- Col Attention: 沿高度方向做自注意力
- 效率高于全局自注意力(O(N²)→O(N√N))

**FeedForward(双门控前馈网络)**:
```python
x1, x2 = dwconv(project_in(x)).chunk(2, dim=1)
x = GELU(x2) * x1 + GELU(x1) * x2    # 双门控机制
```
比标准FFN更强的非线性表达能力。

### 4.3 图像修复模块 — `models/refine.py`

`Refiner` 是一个**多尺度编解码器**，接收5通道输入（3通道RGB原图+2通道阴影掩膜），输出3通道无阴影图像。

```python
class Refiner(nn.Module):
    def __init__(self):
        dim = 16
        self.patch_embed = OverlapPatchEmbed(5, dim)    # 5通道→16通道

        # 编码器：三级下采样（dim→2dim→4dim→8dim）
        self.down_1 = Downsample(dim)       # 空间/2, 通道×2
        self.down_2 = Downsample(dim*2)     # 空间/4, 通道×4
        self.down_3 = Downsample(dim*4)     # 空间/8, 通道×8

        # 编码器每级：两个空洞卷积 + 聚合模块
        self.block_down_1_1, self.block_down_1_2  # dilation=8 (大视野)
        self.aggreation_down_1                      # 通道注意力融合
        self.block_down_2_1, self.block_down_2_2  # dilation=4
        self.aggreation_down_2
        self.block_down_3_1, self.block_down_3_2  # dilation=2
        self.aggreation_down_3

        # 瓶颈层：3个NAFBlock（非线性激活自由块）
        self.middle = NAFBlock(c=dim*8) × 3

        # 解码器：三级上采样（对称结构）
        self.up_3 = Upsample(dim*8)         # 空间×2, 通道/2
        self.up_2 = Upsample(dim*4)
        self.up_1 = Upsample(dim*2)

        # 可学习的跳跃连接权重系数 ★
        self.coefficient_3_2 = nn.Parameter(torch.ones((2, dim*4)))
        self.coefficient_2_1 = nn.Parameter(torch.ones((2, dim*2)))
        self.coefficient_1_0 = nn.Parameter(torch.ones((2, dim)))

        # 多尺度空间金字塔池化（SPP）
        self.latent = SPP(dim, dim, num_layers=4)

        # 输出前再经过1个TransformerBlock精细化
        self.refinement = TransformerBlock(dim, ...)

        # 最终输出：Sigmoid激活，值域[0,1]
        self.output = nn.Conv2d(dim, 3, 3, padding=1)
        # forward最后: torch.sigmoid(self.output(out))
```

**Refiner的关键设计**:

1. **空洞卷积级联**: dilation逐级减小(8→4→2)，从大到小感受野捕捉多尺度信息
2. **AGGREGATION模块**: 使用全局平均池化+通道注意力(SE-like)自适应融合特征
3. **可学习跳跃连接**: `coefficient_3_2`等参数自动学习编码器-解码器特征的融合权重
4. **SPP(空间金字塔池化)**: 4层不同尺度池化+插值恢复，捕获全局上下文
5. **NAFBlock**: 非线性激活自由块(来自NAFNet)，使用SimpleGate机制+通道注意力
6. **输出前Transformer**: 最后用1个TransformerBlock进行全局特征精细化

**NAFBlock结构**（来自NAFNet论文）:
```
输入 → LayerNorm2d → Conv1x1 → DWConv3x3 → SimpleGate → 通道注意力(sca) → Conv1x1
     → 残差连接(×beta)
     → LayerNorm2d → Conv1x1 → SimpleGate → Conv1x1
     → 残差连接(×gamma) → 输出
```
- SimpleGate: 将特征分两组逐元素相乘，实现自门控
- beta/gamma: 可学习的残差缩放参数

**下采样/上采样方式**:
- 下采样: Conv + PixelUnshuffle (不同于普通MaxPool，不丢失信息)
- 上采样: Conv + PixelShuffle (亚像素卷积)

### 4.4 关于消融实验

根据论文，消融实验涉及对以下模块的移除/替换：

| 消融变体 | 修改内容 |
|----------|----------|
| 无Otsu | 移除ThresholdFormer中的Otsu阈值计算，仅使用Transformer掩膜 |
| 无Transformer掩膜 | 仅使用Otsu掩膜，不移入Transformer |
| 无跳跃连接 | 移除Refiner中的可学习跳跃连接 |
| 不同Transformer块数 | num_trans_blocks=1,2,3对比 |
| 无SSIM损失 | 仅使用MSE损失训练 |

**要在代码中实现消融实验，可通过以下方式**:
1. 修改`ThresholdFormer.forward()`中`res_mask = torch.cat((otsu_mask, trans_mask), dim=1)`为仅保留一种掩膜
2. 修改`Refiner.forward()`中的跳跃连接为直接相加
3. 修改`Model.__init__()`中`num_trans_blocks`参数
4. 修改`train.py`中`train_loss = loss_psnr + 0.2 * loss_ssim`为`train_loss = loss_psnr`

#### Mixup数据增强（第68-99行）

**这是论文中重要的训练技巧**：

```python
def mixup(self, inp_img, tar_img, mode='mixup'):
    # 随机选择另一张图
    mixup_index_ = random.randint(0, self.sizex - 1)
    _, transformed = self.load(mixup_index_)

    alpha = 0.2
    lam = np.random.beta(alpha, alpha)    # Beta分布采样混合比例

    if mode == 'mixup':
        # 像素级混合
        inp_img = lam * inp_img + (1 - lam) * mixup_inp_img
        tar_img = lam * tar_img + (1 - lam) * mixup_tar_img
    else:  # mode == 'cutmix'
        # 区域级替换
        # 随机选择一个矩形区域，用另一张图对应区域替换
        inp_img[:, y0:y1, x0:x1] = mixup_inp_img[:, y0:y1, x0:x1]
        tar_img[:, y0:y1, x0:x1] = mixup_tar_img[:, y0:y1, x0:x1]
```

**Mixup触发条件**:
- 仅在训练模式下
- `index > 0 且 index % 3 == 0`的样本（每3个样本触发1次）
- 随机选择mixup或cutmix模式

```python
    tar_img = F.to_tensor(transformed['target'])

            ...

    bin_img = F.rgb_to_grayscale(inp_img)            # 转为灰度图
```

```python
def seed_everything(seed=3407):
    """固定所有随机种子，确保实验可复现"""
    os.environ['PYTHONHASHSEED'] = str(seed)
    random.seed(seed)
    np.random.seed(seed)
    torch.manual_seed(seed)
    torch.cuda.manual_seed(seed)
    torch.cuda.manual_seed_all(seed)
    torch.backends.cudnn.deterministic = True

```
- `cudnn.deterministic = True`: 使用确定性算法（速度略慢但可复现）
- `cudnn.benchmark = False`: 不自动寻找最优卷积算法

```python
def save_checkpoint(state, epoch, model_name, outdir):
    """保存检查点: 命名格式为 {model_name}_epoch_{epoch}.pth"""
    checkpoint_file = os.path.join(outdir, model_name + '_' + 'epoch_' + str(epoch) + '.pth')
    torch.save(state, checkpoint_file)

def load_checkpoint(model, weights):
    """加载检查点，自动处理DataParallel的'module.'前缀"""
    checkpoint = torch.load(weights, map_location=...)
    new_state_dict = OrderedDict()
    for key, value in checkpoint['state_dict'].items():
        if key.startswith('module'):
            name = key[7:]       # 移除DataParallel包装的 'module.' 前缀
        else:
            name = key
        new_state_dict[name] = value
    model.load_state_dict(new_state_dict)
```
